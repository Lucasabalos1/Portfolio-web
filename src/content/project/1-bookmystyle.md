## Sobre el proyecto
Sistema web para la gestión de turnos de una peluquería. Es mi primer proyecto Fullstack desarrollado íntegramente en solitario, donde integré el frontend y el backend por mi cuenta.

### Tecnologías utilizadas
- **React** como framework para el frontend.
- **Python** como lenguaje para el backend.
- **Javascript** como lenguaje para el frontend.
- **Sqlite3** como base de datos

### Modulos 
- **Modulo de logueo** Este módulo está centrado en la administración del logueo al sistema, en donde se verifican las  credenciales de los empleados para poder acceder a la página.
- **Módulo de gestión de turnos** Este módulo está encargado de la gestión de turnos, en donde el empleado podrá crear un nuevo turno y asignarle un cliente, este podrá marcarlo como terminado, editarlo o cancelarlo.
- **Módulo de gestión de clientes** Este módulo se encarga en la gestión de clientes, es decir, se encarga de registrar a los clientes para que posteriormente queden cargados en el sistema y pueda asignarles un turno.
- **Módulo de gestión de empleados** Este módulo está centrado en la gestión de empleados, en donde se podrá dar de alta a los empleados ya sea como empleado o administrador.
- **Módulo de gestión de métricas** Este módulo está encargado de la gestión de métricas, en donde el administrador podrá ver gráficos con datos relevantes.
- **Módulo de gestión de servicios** Este módulo está encargado de la gestión de servicios, el usuario podrá crear, editar y eliminar los servicios de la peluquería.
- **Módulo de gestión de atención** Este módulo está encargado de la gestión de atención a los clientes, en donde se podrá configurar los días y horas en la que se podrá asignar un turno.
