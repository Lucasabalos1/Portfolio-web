## Sobre el proyecto
Este es un proyecto desarrollado íntegramente con tecnologías web puras (Vanilla JS). El objetivo fue crear una plataforma funcional que permita a los usuarios explorar catálogos de anime, gestionar sus propias listas y calificar sus series favoritas. 

La aplicación gestiona el estado del usuario y las listas de forma local, permitiendo una experiencia fluida de navegación y búsqueda sin dependencias de frameworks externos.

### Características principales
- **Gestión de Usuarios:** Sistema de login funcional para acceder a funciones personalizadas.
- **Exploración y Top Global:** Visualización de los animes mejor rankeados y acceso a la lista completa de la comunidad.
- **Buscador Avanzado:** Página dedicada para filtrar y encontrar series específicas.
- **Detalle Dinámico:** Ficha técnica de cada anime con opción de calificación personalizada para usuarios registrados.
- **Generador Aleatorio:** Función para descubrir nuevos animes al azar.

### Tecnologías utilizadas
- **JavaScript (ES6+):** Lógica de manipulación del DOM, manejo de eventos y gestión de datos en el cliente.
- **HTML5 & CSS3:** Estructura semántica y diseño de interfaz personalizado sin librerías externas.
- **LocalStorage:** Persistencia de datos para mantener las listas de los usuarios y sesiones de forma local.
